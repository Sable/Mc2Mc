# Mc2Mc

A MATLAB to MATLAB source code translator.

## Configuration

Platforms:

- Tamer framework (Developed by Sable/McGill)
- IntelliJ IDEA (tested under v15.04)

## Options

```
  Options:
    -a, --all
       Enable all optimization
       Default: true
    -d, --disable
       Disable all optimization
       Default: false
    -p, --display
       list all optimizations
       Default: false
    -e, --enable
       Enable all optimization
       Default: true
    -h, --help
       list all commands
       Default: false
```
